Описание модели

Система предназначена для оптимального выбора средств защиты информации (СЗИ) при заданных бюджетных ограничениях для различных групп активов (ГИА). Модель минимизирует риски, балансируя между суммарным временем взлома и временем реакции самого слабого звена через коэффициент λ.

Параметры ввода
m Количество групп активов (строки матрицы A)
n Количество доступных СЗИ (столбцы)
λ Приоритет: 1.0 = фокус на времени взлома, 0.0 = фокус на реакции
Матрица A Стоимость каждого СЗИ для каждой ГИА (m × n)
Вектор b Бюджетный лимит для каждой ГИА
Вектор c Время взлома для каждого СЗИ
Вектор d Время реакции для каждого СЗИ
Формат файла импорта (.txt)

Каждая строка — одна запись, значения разделяются пробелами:

m n
A[0,0] A[0,1] ... A[0,n-1]
A[1,0] A[1,1] ... A[1,n-1]
...
b[0] b[1] ... b[m-1]
c[0] c[1] ... c[n-1]
d[0] d[1] ... d[n-1]
lambda
Пример файла (m=3, n=4) 
3 4
10 20 15 25
12 18 22 10
8  16 20 30
50 60 55
30 40 35 45
5  8  6  9
0.5
