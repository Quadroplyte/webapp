Документация и руководство

Описание модели

Данная система предназначена для оптимального выбора средств защиты информации (СЗИ) при заданных бюджетных ограничениях для различных групп активов (ГИА).

Математическая постановка: Модель минимизирует риски, балансируя между суммарным временем взлома (аддитивный критерий) и временем реакции самого слабого звена (максиминный критерий) с помощью коэффициента λ.

Параметры ввода

  • m — количество групп активов (строки матрицы ограничений).
  • n — количество доступных СЗИ (столбцы).
  • λ (Lambda) — приоритет. 1.0 — фокус на общем времени защиты, 0.0 — на скорости реакции.
  • Матрица A — стоимость внедрения СЗИ. Каждая ячейка A[i,j] — цена j-го СЗИ для i-й группы.
  • Вектор b — лимит бюджета для каждой группы ГИА.
  • Вектор c — время взлома для каждого СЗИ.
  • Вектор d — время реакции для каждого СЗИ.

Импорт из файла (input.txt)

Файл должен быть в текстовом формате (.txt) со значениями через пробел:

m n
A (матрица m x n)
b (вектор m)
c (вектор n)
d (вектор n)
lambda
