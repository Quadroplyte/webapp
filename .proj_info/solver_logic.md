# Алгоритм решения задачи (solver.py)

Алгоритм работает с порядком данных, в котором они были записаны в исходном условии.

**Алгоритм**
1. Пусть $V$ — пустое множество. Решить задачу однокритериальную многомерную задачу о ранце с аддитивным критерием (используя PuLP) и получить оптимальное решение — $n$-мерный булевый вектор $x^0$; поместим его в множество $V$. Если $\lambda = 1$, то вектор $x^0$ будет оптимальным решением. В этом случае алгоритм окончен.
2. Далее считаем, что $\lambda \in (0, 1)$. Пусть $j_0$ — номер в полученном оптимальном решении $x^0$, для которого
$$ j_0 = \max \{ j : x_j^0 = 1, j = \overline{1,n} \}, $$
то есть $x_j^0 = 0$, $j = j_0 + 1, \dots, n$.
3. **Набор из $j_0 - 1$ задач.** Решить последовательно $j_0 - 1$ задач о ранце.
Число СЗИ в $s$-ой задаче: $j_0 - s$. Имеются в виду СЗИ с номерами от 1 до $j_0 - s$.
Финансовые ограничения принимают вид
$$ A_s x \le b, $$
где матрица $A_s$ размером $m \times (j_0 - s)$ получена из матрицы $A$ вычеркиванием последних $n - j_0 + s$ столбцов. Введем критерии
$$ F^{(s)}(x) = \lambda \sum_{j=1}^{j_0-s} c_j x_j + (1 - \lambda) d_{j_0-s} x_{j_0-s} = \sum_{j=1}^{j_0-s-1} \lambda c_j x_j + (\lambda c_{j_0-s} + (1 - \lambda) d_{j_0-s})x_{j_0-s} \to \max, \quad s = \overline{1, j_0 - 1}. $$
При фиксированном $s$ данный критерий является аддитивным.
Для всех натуральных $s$, $s = \overline{1, j_0 - 1}$, решить задачу о ранце (всего $j_0 - 1$ задачу), то есть получить оптимальное решение — $(j_0 - s)$-мерный булевый вектор $x^s$. Привести его к $n$-мерному булеву вектору, дополнив отсутствующие компоненты справа нулями, и поместить в множество $V$. Оставить прежнее обозначение $x^s$. Нетрудно видеть, что при этом значение $F^{(s)}(x^s)$ не изменится.
4. В итоге множество $V$ будет содержать $j_0$ $n$-мерных векторов $x^s$, $s = 0, \dots, j_0 - 1$, $j_0 \le n$. Методом полного перебора выбрать из них оптимальный вектор для итогового критерия:
$$F(x) = \lambda \sum_{j=1}^{n} c_j x_j + (1 - \lambda) \min_{x_j=1} d_j \to \max.$$
Этот алгоритм также подходит для решения скалярной задачи о ранце.
