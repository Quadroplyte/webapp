# SZI/IST Knapsack Solver

Проект предназначен для решения задачи оптимального выбора средств защиты информации (СЗИ / IST) с использованием методов математического программирования.

## Основные возможности
- Решение многокритериальной задачи о рюкзаке (Knapsack Problem) без предварительной сортировки.
- Веб-интерфейс с разделением результатов на «Лучшее решение» и полный список «Кандидатов».
- Современный glassmorphic дизайн с трёхцветными градиентными фонами и стеклянными UI-компонентами.
- Стеклянные pill-селекторы для навигации и выбора языка со скользящим индикатором.
- Glass Dropdown для выбора темы оформления (Светлая / Тёмная) прямо в настройках.
- Стеклянный тумблер (toggle) в настройках для **Режима производительности** — отключает все `backdrop-filter` и прозрачности для ускорения отрисовки.
- Выбор орнамента фона: **Точки**, **Плюсики**, **Крестики**, **Нет**.
- **Нейтральные акценты**: в светлой теме убраны синие акценты (кнопки, выделения — нейтральные), в тёмной убраны жёлтые акценты. Все кнопки в одной теме имеют единый вид.
- Подсветка карточки лучшего решения: **розово-голубое звёздное свечение** в светлой теме, золотое — в тёмной.
- Дианамическая адаптивность (Mobile View): на мобильных устройствах `<1024px` колонки собираются в стопку, а панель навигации скрывается и фиксируется снизу.
- Возможность закрепить навбар в трех режимах (Справа статичный, По центру плавающий, Внизу платформой в моб виде).
- Динамическое выделение цветом выбранных СЗИ и лучшего результата.
- Возможность запуска как веб-сервера (FastAPI) или как настольного приложения.
- Поддержка тёмной и светлой темы с сохранением настроек `localStorage`.
- Двуязычный интерфейс (RU / EN), в том числе раздельные подписи для ПК и телефонов.
- Поддержка импорта данных из файлов с автоматической проверкой форматов.

## Визуальный стиль
- **Фон**: Трёхцветный радиальный градиент от нижнего правого угла. Светлая тема: голубой → розовый → серый. Тёмная тема: янтарный → тёмно-синий → чёрный. Поверх накладывается орнамент (Точки / Плюсики / Крестики / Нет).
- **Компоненты**: Glassmorphism — полупрозрачные элементы с `backdrop-filter: blur()`, тонкими границами и внутренними тенями.
- **Навигация**: Glass pill контейнеры со скользящей стеклянной плашкой-индикатором.
- **Выбор темы**: Glass Dropdown в настройках.
- **Карточки**: Frosted glass карточки на страницах настроек и справки.
- **Кнопки**: Все кнопки действий в одной теме выглядят одинаково (единый нейтральный стиль).
- **Режим производительности**: При включении тумблера убираются все эффекты размытия и прозрачности.

## Технологический стек
- **Backend:** Python, FastAPI, Uvicorn, PuLP (линейное программирование), NumPy.
- **Frontend:** Vanilla JS, JSON-based localization (`lang.js`), HTML5, модульный CSS3 (5 файлов: variables, layout, glass, components, pages).
- **Desktop:** pywebview, PyInstaller.
- **Deployment:** Railway (для сервера), PyInstaller (для Windows EXE).

## Как запустить
### Локальный сервер
```bash
python app.py
```
Сервер будет доступен по адресу `http://127.0.0.1:8000`.

### Настольное приложение (Desktop)
```bash
python main_window.py
```

### Сборка исполняемого файла (.exe)
Запустите `build_exe.bat`. Артефакты сборки появятся в папке `dist/`.
