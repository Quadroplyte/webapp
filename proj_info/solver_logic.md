# Алгоритм решения задачи (solver.py)

Ядро проекта (`solver.py`) решает задачу оптимального выбора СЗИ, объединяя два ключевых критерия:
1. **Суммарное время взлома системы** (аддитивный критерий, вектор `c`).
2. **Время реакции самого слабого звена** (минимальное время среди выбранных СЗИ, максиминный критерий, вектор `d`).

## Шаги алгоритма:
1. **Предварительная сортировка:** Все входящие данные (СЗИ) сортируются по убыванию значения критерия `d` (времени реакции).
2. **Шаг 1 (Базовое решение):** Решается стандартная задача о рюкзаке с использованием только критерия `c` и бюджетных ограничений `A x <= b` методом целочисленного линейного программирования (`pulp`).
3. **Шаг 2 (Порог):** Определяется "глубина" поиска кандидатов на основе базового решения.
4. **Шаг 3 (Генерация кандидатов):** Если λ ≠ 1, алгоритм генерирует дополнительные варианты решений, варьируя состав СЗИ и находя компромисс между `c` и `d`.
5. **Шаг 4 (Выбор лучшего):** Из всех найденных вариантов выбирается решение, максимизирующее взвешенную функцию:
   `F = λ * (Σ c_j * x_j) + (1 - λ) * min(d_j | x_j = 1)`

## Используемые библиотеки
- **PuLP:** Библиотека для моделирования задач линейного программирования. Использует стандартный солвер CBC.
- **NumPy:** Быстрые операции над массивами и матрицами.
