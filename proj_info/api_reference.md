# API Reference

Бэкенд предоставляет следующие эндпоинты.

## `GET /`
Возвращает основную HTML-страницу (`static/index.html`).

## `POST /api/solve`
Принимает JSON с параметрами задачи и возвращает найденное решение.

### Параметры тела запроса (SolveRequest):
- `m` (int): Количество ограничений (групп активов).
- `n` (int): Количество доступных СЗИ.
- `lam` (float): Коэффициент λ ∈ [0, 1] для взвешивания критериев.
- `A` (List[List[float]]): Матрица стоимости (m × n).
- `b` (List[float]): Вектор бюджетных ограничений (длина m).
- `c` (List[float]): Индивидуальное время взлома для каждого СЗИ (длина n).
- `d` (List[float]): Время реакции для каждого СЗИ (длина n).

### Ответ:
- `success` (bool): Флаг успешного завершения.
- `vector_x` (List[int]): Бинарный вектор выбранных СЗИ (длина n).
- `optimal_f` (float): Значение комбинированной целевой функции.
- `recommended_szi` (List[int]): Список порядковых номеров (с 1) выбранных СЗИ.
- `error` (str, опционально): Описание ошибки при `success: false`.
