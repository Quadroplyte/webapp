# SZI Knapsack Solver

Проект предназначен для решения задачи оптимального выбора средств защиты информации (СЗИ) с использованием методов математического программирования.

## Основные возможности
- Решение многокритериальной задачи о рюкзаке (Knapsack Problem).
- Веб-интерфейс для ввода данных и визуализации результатов.
- Возможность запуска как веб-сервера (FastAPI) или как настольного приложения (pywebview + PyInstaller).
- Поддержка темной и светлой темы с автоматическим сохранением настроек.
- Интерактивный дашборд с разделами Оптимизации, Документации и Настроек.
- Поддержка деплоя на Railway.

## Технологический стек
- **Backend:** Python, FastAPI, Uvicorn, PuLP (линейное программирование), NumPy.
- **Frontend:** Vanilla JS, HTML5, CSS3.
- **Desktop:** pywebview, PyInstaller.
- **Deployment:** Railway (для сервера), PyInstaller (для Windows EXE).

## Как запустить
### Локальный сервер
```bash
python app.py
```
Сервер будет доступен по адресу `http://127.0.0.1:8000`.

### Настольное приложение (Desktop)
```bash
python main_window.py
```

### Сборка исполняемого файла (.exe)
Запустите `build_exe.bat`. Артефакты сборки появятся в папке `dist/`.
